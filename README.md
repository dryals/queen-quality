# queen-quality
GS and GWAS on queen size and fertility traits

Dylan Ryals, Bradley Metz, Luiz Brito, David Tarpy, Brock Harpur

The script `1_data_prep.sh` is designed for SLURM protocol on an HPC and generates all data using scripts found in the `scripts` directory. The R markdown file `2_analysis.Rmd` runs analyses and generates figures.
